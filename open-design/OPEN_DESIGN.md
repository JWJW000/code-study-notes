# Open Design Integration

This project uses design tokens and component conventions adapted from [`nexu-io/open-design`](https://github.com/nexu-io/open-design).

Source design system:

```text
open-design/design-systems/vercel/tokens.css
open-design/design-systems/vercel/components.html
```

The Web UI keeps the Python standard-library server, but imports the selected Open Design token block directly into the generated HTML/CSS so the artifact is traceable to Open Design rather than only being visually inspired.
